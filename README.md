## Requirements
* PHP 7+
* NodeJS
* Composer
* Laravel 8 With Jetstream and Livewire installed

## Installation
* Edit .env file according to your needs
* Run :
```bash
$ composer update
$ php artisan classroom:setup
```

## Usage
Just register and login
